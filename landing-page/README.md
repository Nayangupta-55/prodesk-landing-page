#  Prodesk IT Landing Page

A responsive and modern landing page built as part of my Sprint submission.

## Features

* Fully responsive (Mobile + Desktop)
* Dark/Light mode toggle
* Sticky glassmorphism navbar
* Interactive service cards
* Clean UI with modern color palette

## Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* Vanilla JavaScript
